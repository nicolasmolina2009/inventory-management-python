# Inventory Management Python

A simple open-source Python project for basic inventory management.

This project was created as part of my learning path in Systems Engineering and open-source development. It is focused on managing products, stock quantities, purchase costs, sale prices, and basic inventory movements.

## Purpose

The purpose of this project is to practice software development with Python while building a useful tool related to inventory and warehouse management.

## Features

- Register products with name, category, quantity, cost, and sale price.
- List all products in the inventory.
- Search products by name.
- Add stock entries.
- Register stock outputs.
- Calculate basic profit margin.
- Store inventory data in a JSON file.

## Technologies

- Python 3
- JSON file storage
- Command-line interface

## How to run

1. Clone the repository:

```bash
git clone https://github.com/nicolasmolina2009/inventory-management-python.git
```

2. Enter the project folder:

```bash
cd inventory-management-python
```

3. Run the program:

```bash
python main.py
```
